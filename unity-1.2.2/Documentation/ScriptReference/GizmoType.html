enum GizmoType
Overview | Class Hierarchy | Alphabetical Reference

Determines how the gizmo is drawn.

Values

Active

SelectedOrChild

NotSelected

Selected

Pickable

See Also: DrawGizmo

Values

GizmoType.Active

Draw the gizmo if it is active? (Shown in the inspector)

GizmoType.SelectedOrChild

Draw the gizmo if it is selected or a child of the selection?

GizmoType.NotSelected

Draw the gizmo even if it is not at all selected?

GizmoType.Selected

Draw the gizmo if it is not selected? It is recommended to always use SelectedOrChild instead.

GizmoType.Pickable

Is it possible to pick the gizmo?
