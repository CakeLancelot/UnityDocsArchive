Trail Renderer

Components > Particle systems > Trail Renderer

The Trail renderer is used to make trails behind objects in the scene as they move about.

Properties

Property:Function:
Materials First element of this array is used for rendering the object. Particle shaders make most sense in this context.
Time Length of the trail, measured in seconds.
Start Width Width of the trail at the object's place.
End Width Width of the trail at the end.
Colors Array of colors to use over the length of the trail.
AutoDestruct Enable this to make the object be destroyed when the object hasn't moved for Time seconds

Details

The TrailRenderer is great for a trail behind a missile, or contrails from the tip of a plane's wings. It is good to add a general feeling of speed.

When using a TrailRenderer, no other filters in the Pipeline are used.

Hints

{HINT ()} Use particle materials with the Trail Renderer. {HINT}
