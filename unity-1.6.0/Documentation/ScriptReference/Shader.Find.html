Usage

Shader Find (string name)

Description

Finds a shader with the given name.

Shader.Find can be used to switch to another shader without having to keep a reference to the shader. name is the name you can see in the shader popup of any material. Common names are: " Diffuse", " Bumped", " Glossy", "Alpha/Diffuse" etc.

// Switch the shader from code function Start ()
{
// Switch to the alpha diffuse shader renderer.material.shader = Shader.Find("Alpha/Diffuse");
}

// Create a material from code function Start ()
{
// Create a material with alpha diffuse shader var material = new Material (Shader.Find("Alpha/Diffuse"));
material.color = Color.green;
// assign the material to the renderer renderer.material = material;
}
