This class opens up all properties from a material, allowing you to animate them. in order to get a material from an object, use the Renderer's material property.

Material.Material

Material (string contents)

Create a temporary material. If you have a script which implements a custom special effect, you implement all the graphic setup using shaders & materials. Use this function to create a custom shader & material inside your script. After creating the material, use SetColor, SetTexture to populate the values.

Methods

Material.SetColor

void SetColor (string propertyName, Color color)

Set a named color value. Many shaders have more than one color. Use SetColor to change the propertyName color. "_Color" is the main color of a material. "_SpecColor" is the specular color of a material. "_Emission" is the emissive color of a material "_ReflectColor" is the reflection color of the material.

Material.GetColor

Color GetColor (string propertyName)

Get a named color value. Use GetColor to get the propertyName color from the material.

Material.Lerp

void Lerp (Material start, Material end, float t)

Lerp all properties between two material. Makes all color and float values of a material be interpolated from start to end, based on t. If t is 0, all values are taken from start. if t is 1, all values are taken from end.
