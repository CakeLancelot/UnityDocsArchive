GUILayer.HitTest

GUIElement HitTest (Vector3 screenPosition)

Get the GUI element at a specific screen position. Returns the GUIElement at a specific point on screen. If screenPosition is inside a GUIElement, That one is returned. If not, this function return NULL. screenPosition is measured in screen coordinates, like the values returned by Input's mousePosition property.

See Also: GUIElement.HitTest, Input.mousePosition
