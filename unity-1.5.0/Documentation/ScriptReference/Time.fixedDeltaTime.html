Time.fixedDeltaTime

float fixedDeltaTime

The interval at which physics and other fixed frame rate updates (like MonoBehaviour's FixedUpdate) are performed.

It is recommended to use Time.deltaTime instead because it automatically returns the right delta time if you are inside a FixedUpdate function or Update function.
