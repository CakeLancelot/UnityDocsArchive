Mesh Morpher

Components > Meshes > Mesh Morpher

The Mesh morpher filter takes a number of meshes and morphs between them over time.

Properties

Property:Function:
Enabled If enabled, the Mesh Morpher cycles between the different frames automatically.

Speed

Details

The Mesh Morpher works a lot like the Lod Mesh Filter. You want to have a Mesh Renderer as the other filter in the object.

Hints

  • You can animate the Frame number to modify the speed of the animation.
