DrawGizmo.DrawGizmo

DrawGizmo (GizmoType gizmo)

Defines when the gizmo should be invoked for drawing. See Also: GizmoType

DrawGizmo (GizmoType gizmo, Type drawnGizmoType)

Same as above. drawnGizmoType determines of what type the object we are drawing the gizmo of has to be. If drawnGizmoType is null, the type will be determined from the first parameter of the draw gizmo method.
