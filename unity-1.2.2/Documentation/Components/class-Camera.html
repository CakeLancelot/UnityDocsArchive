Camera

Components > Rendering > Camera

Cameras are the devices which the player views the world through.

You can have an unlimited number of cameras in a scene, and arrange and stack them using the View Size and Depth values, described below.

Properties

Property:Function:
Field Of view How wide the camera view angle is measured in degrees along the vertical axis
Background color The clear color of the camera. Only has an effect if Color is enabled in Clearing
Clearing Clearing decides what should be cleared
Color
clear the background color. Disable this if you have a skybox or other fullscreen effect in the background Depth
Clipping The distance to the near and far clipping planes.
View size The size of the camera's viewport. Use this to limit the camera to a certain part of the screen. All values are between 0-1
Depth The camera depth in the draw order. Cameras with the lowest numbers are drawn first.

Hints

  • Use multiple cameras to increase the coolness of your game: try putting one in the tip of a player missile. Place it in one corner of the screnn and give it a higher depth value than the main camera.
  • To increase the sense of speed in a racing game, use a high field of view.
