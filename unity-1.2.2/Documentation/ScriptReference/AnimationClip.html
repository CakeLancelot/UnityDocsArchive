class AnimationClip
Overview | Class Hierarchy | Alphabetical Reference

Object

Stores keyframe based animations.

Variables

length

AnimationClip is used by Animation to play back animations.

Variables

AnimationClip.length

float length

The length in seconds the animation takes to play back.

animation.Play(animation.clip.length);
// Wait for the animation to have finished
yield WaitForSeconds (animation.clip.length);
