How Do I Make a Cube Map?

Unity Manual > HOW-TOs > Graphics how-tos > How Do I Make a Cube Map?

Cubemaps are used by all the reflective shaders. Cube maps are one texture that contains six sides. To build one, you either use the Generate Cube Map option in the texture importer settings, or build one from scratch:

  1. Select the folder you want to create the cube map in.
  2. Choose Assets->Create->Cube Map from the main menu, or Create->Cube Map from the project view context menu.
  3. Assign a texture to each of the six faces of the cube map in the inspector.
