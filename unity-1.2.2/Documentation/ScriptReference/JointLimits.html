JointLimits is used by the HingeJoint to limit the joints angle.

Variables

min

minBounce

max

maxBounce

See Also: HingeJoint

Variables

JointLimits.min

float min

The lower limit of the joint. When the joint angle or position is below it, the joint will excert forces to constrain it.

// Set the minimum hinge angle to be 20 degrees
hingeJoint.limits.min = 20;

JointLimits.minBounce

float minBounce

When

Set the joint to bounce off when hitting the lower limit
hingeJoint.limits.minBounce = 1;

JointLimits.max

float max

The upper limit of the joint. When the joint angle or position is above it, the joint will excert forces to constrain it.

// Set the minimum hinge angle to be 20 degrees
hingeJoint.limits.max = 40;

JointLimits.maxBounce

float maxBounce

Set the joint to bounce off when hitting the upper limit
hingeJoint.limits.maxBounce = 1;
