Texture

Render textures can be used to implement image based rendering effects, projectors or surveillance cameras.

Members

Variables

Constructors

Functions

Class Variables

Class Functions

All rendering goes into the active RenderTexture (active class property). If the active RenderTexture is null everything is rendered to the main OpenGL window. Unity Pro only.

Variables

NameDescription
width

The width of the render texture in pixels. Note that unlike Texture's width property, this is both read and write -

height

The height of the render texture in pixels. Note that unlike Texture's height property, this is both read and write -

depth

The depth of the render texture in bits (0, 16, 24, 32 are supported)

isPowerOfTwo

Is the size of the render texture constrained to power of two sizes?
