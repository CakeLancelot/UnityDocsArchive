Stores keyframe based animations.

AnimationClip is used by Animation to play back animations.

Variables
length

The length in seconds the animation takes to play back.

frameRate

The frame rate at which keyframes are sampled

Constructors
AnimationClip

Creates a new animation clip

Functions
SetCurve

Assigns the curve to animate a specific property. If curve is null the curve will be removed. If a curve already exists for that property, it will be replaced.

ClearCurves

Clears all curves from the clip

AddEvent

Adds an animation event to the AnimationClip
