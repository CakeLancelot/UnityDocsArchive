enum ForceMode
Overview | Class Hierarchy | Alphabetical Reference

Option for how to apply a force using Rigidbody.AddForce

Values

Force

VelocityChange

Acceleration

Impulse

Values

ForceMode.Force

Add a force to the rigid body. The higher the mass of a rigidbody the more force you need to move it around. The parameter has unit of mass * distance/ time^2

ForceMode.VelocityChange

Makes the object change it's velocity by the given amount. The applied effect is not dependent on the mass of the object. The parameter has unit of distance / time.

ForceMode.Acceleration

Adds a force to the rigid body factoring out the mass of the body. In contrast to ForceMode.Force it will move the rigidbody the same way regardless of the ridibody's mass. The parameter has unit of distance/ time^2.

ForceMode.Impulse

parameter has unit of mass * distance /time The parameter has unit of mass * distance /time
