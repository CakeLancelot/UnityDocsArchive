Diffuse Fast

Diffuse Fast Properties

This shader is a specialized version of the regular Diffuse shader. The biggest difference is that it is not completely pixel-lit: it does support light cookies, but is more sensitive to tesselation and does not support shadows. If you don't use shadows and don't have lights that are very close to surface, it can be a good idea to try using this shader instead of Diffuse one.

Performance

This shader is a cheaper version of Diffuse shader, that makes it the simplest pixel-lit shader in Unity. For this reason, this shader gives much better performance on low-end cards like GeForce FX5200, Radeon Xpress200 or Intel GMA950.
