Texture2D.GetPixel

Color GetPixel (int x, int y)

Returns pixel color at pixel coordinates (x, y). If the pixel coordinates are out of bounds (larger than width or small than 0), they will be clamped or repeat based on the textures wrap mode

This function works only on RGBA32, RGB24 and Alpha8 texture formats. For other formats it always returns opaque white.

// Sets the y coordinate of the transform to follow the heightmap var heightmap : Texture2D;
var size = Vector3 (100, 10, 100);
function Update ()
{
var x : int = transform.position.x / size.x * heightmap.width;
var z : int = transform.position.z / size.z * heightmap.height;
transform.position.y = heightmap.GetPixel(x, z).grayscale * size.y;
}
