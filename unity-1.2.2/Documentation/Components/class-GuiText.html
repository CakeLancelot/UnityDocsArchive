GUI Text

Components > Rendering > GUI Text

GUI Texts are used to print text onto the screen in 2D. The camera has to have a GUI Layer attached in order to render the text.

GUI Texts are positioned using the x and y components of the Transform of the same game object, where (0,0) is the bottom-left and (1,1) is the top-right corner of the screen

Properties

Property:Function:
Text The text to display.
Anchor How the text is placed relative to the position given in the transform.
Align How multiple lines are aligned within the GUIText.
Line Spacing Extra space to add between lines.
Font The font to use when rendering the text.
Material The material containing the characters to be drawn. This property overrides the one in the Font asset
