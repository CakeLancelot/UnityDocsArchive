Behaviour

Interface for light components.

Members

Variables

Functions

Class Variables

Class Functions

Use this to control all aspects of Unity's lights. The properties are an exact match for the values shown in the inspector.

Usually lights are just create in the editor but sometimes you want to create a light from a script:

function Start ()
{
// Make a game object var lightGameObject = new GameObject("The Light");
// Add the light lightGameObject.AddComponent(Light);
// lightGameObject.light.color = Color.blue;
lightGameObject.transform.position = Vector3 (0, 5, 0);
}
