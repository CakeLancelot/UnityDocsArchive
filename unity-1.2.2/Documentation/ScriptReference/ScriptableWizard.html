class ScriptableWizard
Overview | Class Hierarchy | Alphabetical Reference

ScriptableObject

This is an editor class. To use it you have to place your script in Assets/Editor inside your project folder. Editor classes are in the UnityEditor namespace so you need to add "import UnityEditor;" in the beginning of the script.

Derive from this class to create a wizard.

ScriptableWizard.OnWizardUpdate

string OnWizardUpdate ()

This is called whenever the user changes something in the inspector and when the wizard is opened. This allows you to set the helpString, errorString and enable/disable the create button use isValid

See Also: ScriptableWizard.DisplayWizard

void OnWizardUpdate () {
    
    helpString = "Please set the color of the light!";

    // Disable the wizard create button
    isValid = false;
    // And tell the user why
    errorString = "You absolutely must set the color of the light!";
}

ScriptableWizard.OnWizardCreate

bool OnWizardCreate ()

Allows you to provide a Create function. This is called when the user clicks on the Create button. After the Create function is called, the wizard is automatically closed.

See Also: ScriptableWizard.DisplayWizard

Class Methods

ScriptableWizard.DisplayWizard

void DisplayWizard (string title, Type klass, string createButtonName = "Create", string otherButtonName = "")

Creates a wizard with title. The klass has to derive from ScriptableObject. When the user hits the Create button the OnWizardCreate function will be called. DisplayWizard will only show one wizard for every wizard class (eg. WizardCreateLight in the example below)

// C# example:
using UnityEditor;
using UnityEngine;
class WizardCreateLight : ScriptableWizard {
public float range = 500;
public Color color = Color.red;

[MenuItem ("GameObject/Create Light Wizard")]
static void CreateWizard () {
        ScriptableWizard.DisplayWizard("Create Light", typeof (WizardCreateLight),
            "Create", "Apply");

        //If you don't want to use the secondary button simply leave it out:
        //ScriptableWizard.DisplayWizard("Create Light", typeof(WizardCreateLight));
}

void OnWizardCreate () {
GameObject go = new GameObject ("New Light");
go.AddComponent("Light");
go.light.range = range;
go.light.color = color;
}

void OnWizardUpdate () {
helpString = "Please set the color of the light!";
}

// When the user pressed the "Apply" button OnWizardOtherButton is called.
void OnWizardOtherButton ()
{
// Simply set the color of the selected light to red.
if (Selection.activeTransform == null)
return;
if (Selection.activeTransform.light == null)
return;
Selection.activeTransform.light.color = Color.red;
}
}
