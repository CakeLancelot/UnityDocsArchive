class AudioListener
Overview | Class Hierarchy | Alphabetical Reference

Representation of a listener in 3D space.

Class Variables

volume

pause

This class implements a microphone-like device. It records the sounds around it and plays that through the player's speakers. You can only have one listener in a scene.

See Also: AudioSource, AudioListener in the Components Reference.

Class Variables

AudioListener.volume

float volume

Controls the game sound volume.

AudioListener.volume = 0.5;

AudioListener.pause

bool pause

The paused state of the audio. If set to True, the listener will not generate sound. Similar to setting the volume to 0.0.

AudioListener.pause = true;
