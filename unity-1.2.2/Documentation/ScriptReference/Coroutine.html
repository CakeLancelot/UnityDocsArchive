YieldInstruction

MonoBehaviour's StartCoroutine method returns Coroutine.

The coroutine can be used to yield until the Started coroutine is finished.

// - prints "Starting 0.0"
// - prints "WaitAndPrint 5.0"
// - prints "Done 5.0"
print ("Starting " + Time.time);
yield StartCoroutine ("WaitAndPrint");
print ("Done " + Time.time);

function WaitAndPrint () {
    yield WaitForSeconds (5);
    print ("WaitAndPrint "+ Time.time);
}
