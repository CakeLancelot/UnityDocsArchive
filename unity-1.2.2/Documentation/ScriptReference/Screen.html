Class Variables

Screen.resolutions

Resolution[] resolutions

Returns a list of all supported fullscreen resolutions by the monitor. The list is sorted by width. Low resolutions come first.

var resolutions : Resolution[] = Screen.resolutions;

for (var res in resolutions)
{
    print(res.width);
    print(res.height);
}
// Switch to the lowest supported resolution
Screen.SetResolution (resolutions[0].width, resolutions[0].height, true);

Screen.showCursor

bool showCursor

Should the cursor be visible? The cursor will only be hidden in fullscreen mode.

// Hide the cursor
Screen.showCursor = false;

Class Methods

Screen.SetResolution

void SetResolution (int width, int height, bool fullscreen, int preferredRefreshRate = 0)

Switches the resolution. If no matching resolution can be found the closest matching one will be used. If no preferredRefreshRate is 0 (default) unity will switch to the highest refresh rate supported by the monitor. If preferredRefreshRate is not 0 unity will use preferredRefreshRate if the monitor supports it.

// Switch resolution to 640 x 480 full screen
Screen.SetResolution (640, 480, true);
