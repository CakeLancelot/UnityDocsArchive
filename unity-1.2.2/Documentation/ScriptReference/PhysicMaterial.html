Variables

PhysicMaterial.dynamicFriction

float dynamicFriction

The friction used when already moving. This value has to be between 0 and 1. A value of 0 feels like ice, 1 feels like rubber.

collider.dynamicFriction = 1;

PhysicMaterial.staticFriction

float staticFriction

The friction used when an object is lying on a surface. Usually a value from 0 to 1. A value of 0 feels like ice, 1 feels like rubber.

collider.staticFriction = 1;

PhysicMaterial.bouncyness

float bouncyness

How bouncy is the surface? A value of 0 will not bounce. A value of 1 will bounce without any loss of energy.

collider.bouncyness = 1;

PhysicMaterial.frictionDirection2

Vector3 frictionDirection2

The direction of anisotropy. Anisotropic friction is enabled if the Vector3 is not zero. dynamicFriction2 and staticFriction2 will be applied along frictionDirection2. The direction of anisotropy is relative to the colliders local coordinate system.

// Make the collider slippery when going forward but not when going sideways
collider.physicMaterial.frictionDirection2 = Vector3.forward;
collider.physicMaterial.dynamicFriction2 = 0;
collider.physicMaterial.dynamicFriction = 1;

PhysicMaterial.dynamicFriction2

float dynamicFriction2

If anisotropic friction is enabled, dynamicFriction2 will be applied along frictionDirection2.

collider.physicMaterial.dynamicFriction2 = 0;

PhysicMaterial.staticFriction2

float staticFriction2

If anisotropic friction is enabled, staticFriction2 will be applied along frictionDirection2.

collider.physicMaterial.staticFriction2 = 0;

PhysicMaterial.useSpring

bool useSpring

Should the surface of the collider be springy?

collider.material.useSpring = true;

PhysicMaterial.spring

JointSpring spring

This is useful for colliders used as wheels are other materials that should appear wobbly. The spring force is spring.spring. The damping force is spring.damping. The damping force is spring.spring. spring.damper is the damping force. spring.damper is the damping force. The spring parameters for the surface. The spring attempts to reach a target distance by adding spring and damping forces.

The spring.spring force attempts to reach the target distance. A larger value makes the spring reach the target position faster.

The spring.damper force dampens the angular velocity. A larger value makes the spring reach the goal slower.

The spring reaches for the spring.targetPosition. If spring.targetPosition is zero the spring will rest when the surfaces are just touch each other. collider.material.

PhysicMaterial.frictionCombine

PhysicMaterialCombine frictionCombine

Determines how the friction is combined. Traditionally friction properties are dependent on the combination of the two materials in contact. This is however impractical in a game. Instead you can use the combine mode to tune how the friction values of two materials are combined.

collider.material.frictionCombine = FrictionCombineMode.Average;

PhysicMaterial.bounceCombine

PhysicMaterialCombine bounceCombine

Determines how the bouncyness is combined. Traditionally bouncyness properties are dependent on the combination of the two materials in contact. This is however impractical in a game. Instead you can use the combine mode to tune how the bouncyness values of two materials are combined.

collider.material.bounceCombine = FrictionCombineMode.Average;

Constructors

PhysicMaterial.PhysicMaterial

PhysicMaterial ()

Creates a new material. It is usually easier to just use collider.material and modify the attached material directly.

// Creates a new material and attaches it
function Start () {
    var material = new PhysicMaterial();
    material.dynamicfriction = 1;
    collider.material = material;
}

 
PhysicMaterial (string name)

Creates a new material named name.

// Creates a new material attaches and attaches it
function Start () {
    var material = new PhysicMaterial("New Material");
    material.dynamicfriction = 1;
    collider.material = material;
}
