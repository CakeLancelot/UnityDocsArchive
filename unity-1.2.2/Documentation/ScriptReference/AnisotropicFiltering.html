enum AnisotropicFiltering
Overview | Class Hierarchy | Alphabetical Reference

Used by Texture to set the anisotropic filtering.

Values

Disable

Enable

ForceEnable

Values

AnisotropicFiltering.Disable

Disables anisotropic filtering on all textures.

AnisotropicFiltering.Enable

Uses the anisotropic filtering value specified in the texture. (Default)

AnisotropicFiltering.ForceEnable

enables anisotropic filtering on all textures.
