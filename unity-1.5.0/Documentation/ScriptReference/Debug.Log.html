Debug.Log

void Log (object message)

Logs message to the Unity Console.

Debug.Log ("Hello");

Debug.Log

void Log (object message, Object context)

Logs message to the Unity Console. When you select the error in the console a connection to the context Object will be drawn. This is very useful if you want know on which object an error occurrs.

Debug.Log ("Hello", gameObject);
