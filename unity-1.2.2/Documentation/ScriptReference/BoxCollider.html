Collider

A box-shaped primitive collider.

Variables

center

size

See Also: SphereCollider, CapsuleCollider, PhysicMaterial, Rigidbody

Variables

BoxCollider.center

Vector3 center

The center of the box, measured in the object's local space.

// Reset the center to the transform's position
collider.center = Vector3.zero;

BoxCollider.size

Vector3 size

The size of the box, measured in the object's local space. The box size will be scaled by the transform's scale

// Make the box collider long
collider.size = Vector3(10, 1, 1);
