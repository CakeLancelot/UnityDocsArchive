Quaternion.FromToRotation

Quaternion FromToRotation (Vector3 fromDirection, Vector3 toDirection)

Creates a rotation which rotates from fromDirection to toDirection. Usually you use this to rotate a transform so that one of it's axes eg. the y-axis - follows a target direction toDirection in world space.

// Sets the rotation so that the transform's y-axis follows the surface normal transform.rotation = Quaternion.FromToRotation (Vector3.up, surfaceNormal);
