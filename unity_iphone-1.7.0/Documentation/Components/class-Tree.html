Tree

Tree allows you to create and edit procedural trees. Example trees:


Tree Editor

The Tree Editor is split into several different panels.


Hierachy view

The hierachy view is where you start building your tree. It shows a schematic representation of the tree, where each box is a group of nodes. By selecting one of the groups in the hierachy you can modify it's properties. You may also add or remove groups by clicking one of the buttons displayed immediately above the hierachy.

Add LeavesAdd a leaf group to the currently selected group. Once the leaf group is added, you can modify it from the panel below the hierachy view.
Add BranchesAdd a branch group to the currently selected group. Once the branch group is added, you can modify it from the panel below the hierachy view.
Delete SelectedDeletes the currently selected node, group or spline point.

Tree Editing Tools

While the Tree Editor works with procedural elements, you can at any time decide to modify them by hand to achieve the exact shape you want. By using the Tree Editing Tools, you can quickly and easily adjust the shape. Once you have edited a group by hand, it will be locked so that certain procedural properties are no longer available. You can however always revert a group to be procedural by clicking the button displayed below the editing tools.

MoveSelect a node or spline point in the scene view, and drag it around. Movement always happens on a plane perpendicular to the viewing direction.
RotateSelect a node or spline point in the scene view and drag to rotate it. Rotation always happens around an axis parallel to the viewing direction.
Free Hand DrawingClick on a spline node, and drag mouse to draw a new shape. Release the mouse to complete the drawing. Drawing always happens on a plane perpendicular to the viewing direction.

Tree Properties

Every tree has a root node, which controls global properties for the tree. This is the least complex group type, but it holds some valuable properties that control the rendering and generation of the entire tree.

Distribution (Tree Root Group)

Here you can adjust global properties for the tree. Add parts to the tree in the Hierachy View above.

Tree SeedThe global seed for the tree, changing this will affect the entire tree. Use it to randomize your tree, while keeping the general structure of it.
Area SpreadAdjusts the spreading of trunk nodes, only has an effect if you have more than one trunk. Useful if you are creating a small patch of bamboo as an example.

Geometry (Tree Root Group)

The geometry tab allows you to modify properties related to the geometric construction of your tree.

Ambient OcclusionSelect whether you want the editor to compute ambient occlusion for the tree. While changing a tree property ambient occlusion is always hidden, and won't be computed until you finish the changes. Ambient occlusion can greatly improve the visual quality of your tree, but it does take a short while to compute, so you may disable it until you are happy with your tree.
LOD QualityDefines the level-of-detail for the entire tree. A low value will make the tree less complex, a high value will make the tree more complex. Check the statistics in the hierachy view, to see the current complexity of the mesh. Depending on the type of tree and your target platform you may need to adjust this property, to make the tree fit within your polygon budget. With careful creation of art assets and a little planning you can produce good looking trees with relatively few polygons.

Branch Group Properties

Branch groups generate branch and frond geometry.

Distribution (Branch Group)

Here you can adjust the count of branches in the group and the placement of them. Use the curves to fine tune placement, rotation and scale. The curves are relative to the parent branch, or to the area spread if this is a trunk.

Group SeedThe seed for this group of branches. Modify to vary procedural generation.
FrequencyThe frequency of this group of branches. This adjusts the number of branches created for each parent branch.
DistributionSelect the way the branches are distributed along their parent.
TwirlTwirl around the parent branch.
Whorled StepDefines how many nodes are in each whorled step, when using Whorled distribution. In real plants this is normaly a Fibonacci number.
Growth ScaleDefines the scaling of nodes along the parent node. Use the curve to adjust, and the slider to fade the effect in and out.
Growth AngleDefines the initial angle of growth compared to the parent. Use the curve to adjust, and the slider to fade the effect in and out.

Geometry (Branch Group)

Here you can select what type of geometry is generated for this branch group, and which materials are applied. LOD Quality Multiplier allows you to adjust the quality of this group, relative to the Tree LOD Quality.

LOD MultiplierAllows you to tweak the LOD level of this branch group, so that it is either higher or lower quality than the rest of the tree.
Geometry ModeSelect which type of geometry is generated for this branch group.
Branch MaterialSelect the primary material for the branches.
Break MaterialSelect the material used for capping broken branches.
Frond MaterialSelect the material used for fronds.

Shape (Branch Group)

Here you can adjust the shape and growth of the branches. Use the curves to fine tune the shape, all curves are relative to the branch itself.

LengthAdjusts the length of the branches
RadiusAdjusts the radius of the branches, use the curve to fine-tune the radius along the length of the branches.
Is Length RelativeDetermines whether the radius of the branches are affected by the length
Cap SmoothingMakes the cap/tip of the branches more or less round. Useful for cactii.
GrowthAdjusts the growth of the branches
CrinklynessAdjusts how crinkly/crooked the branches are, use the curve to fine-tune.
Seek Sun/GroundAdjusts how the branches are affected by gravity/sunlight, use the curve to fine-tune.
Surface NoiseAdjusts the surface noise of the branches.
NoiseOverall noise factor, use the curve to fine-tune.
Noise Scale UScale of the noise around the branch, lower values will give a more wobbly look, while higher values gives a more stochastic look.
Noise Scale VScale of the noise along the branch, lower values will give a more wobbly look, while higher values gives a more stochastic look.
FlareDefines flares at the root of the tree. Only valid for trunks.
Flare RadiusThe radius of the flares, this is added to the main radius, so a zero value means no flares.
Flare HeightDefines how far up the trunk the flares start.
Flare NoiseDefines the noise of the flares, lower values will give a more wobbly look, while higher values gives a more stochastic look.
WeldingDefines the welding of branches onto their parent branch. Only valid for secondary branches.
Weld HeightDefines how far up the branch the weld spreading starts.
Weld Spread TopThe spread factor on the top-side of the branch, compared to it's parent branch. A value of zero means no spreading.
Weld Spread BottomThe spread factor on the bottom-side of the branch, compared to it's parent branch. A value of zero means no spreading.

Breaking (Branch Group)

Here you can break branches.

Break ChanceChance of a branch breaking. Ie. 0.0 = no branches are broken, 0.5 = half of the branches are broken, 1.0 = all the branches are broken.
Break LocationThis range defines where the branches will be broken. Relative to the length of the branch.

Fronds (Branch Group)

Here you can adjust the number of fronds and their properties. This tab is only available if you have frond geometry enabled in the Geometry tab.

Frond CountDefines the number of fronds per branch. Fronds are always evenly spaced around the branch.
Frond WidthThe width of the fronds, use the curve to adjust the specific shape along the length of the branch.
Frond RangeDefines the starting and ending point of the fronds.
Frond RotationDefines rotation around the parent branch.
Frond CreaseAdjust to crease / fold the fronds.

Animation (Branch Group)

 Here you can adjust the parameters used for animating this group of branches. In editor mode the wind is always coming from the same direction, and wind zones are not active.
Main WindPrimary wind effect. This creates a soft swaying motion, and is typically the only parameter needed for primary branches.
Main TurbulenceSecondary turbulence effect. Produces more stochastic motion, which is individual per branch. Typically used for branches with fronds, such as ferns and palms.
Edge TurbulenceTurbulence along the edge of fronds. Useful for ferns, palms, etc.

Leaf Group Properties

Leaf groups generate leaf geometry. Either from primitives or from user created meshes.

Distribution (Leaf Group)

Here you can adjust the count of leaves in the group and the placement of them. Use the curves to fine tune placement, rotation and scale. The curves are relative to the parent branch.

Group SeedThe seed for this group of leaves. Modify to vary procedural generation.
FrequencyThe frequency of this group of leaves. This adjusts the number of branches created for each parent branch.
DistributionSelect the way the leaves are distributed along their parent.
TwirlTwirl around the parent branch.
Whorled StepDefines how many nodes are in each whorled step, when using Whorled distribution. In real plants this is normaly a Fibonacci number.
Growth ScaleDefines the scaling of nodes along the parent node. Use the curve to adjust, and the slider to fade the effect in and out.
Growth AngleDefines the initial angle of growth compared to the parent. Use the curve to adjust, and the slider to fade the effect in and out.

Geometry (Leaf Group)

Here you can select what type of geometry is generated for this leaf group, and which materials are applied. If you use a custom mesh, its materials will be used.

Geometry ModeSelect the type of geometry created. You can use a custom mesh, by selecting the Mesh option, ideal for flowers etc.
MaterialThe material used for the leaves.
MeshThe mesh used for the leaves.

Shape (Leaf Group)

Here you can adjust the shape and growth of the leaves.

SizeAdjusts the size of the leaves, use the range to adjust minimum and maximum size.
Perpendicular AlignAdjusts whether the leaves are aligned perpendicular to the parent branch,
Horizontal AlignAdjusts whether the leaves are aligned horizontally.

Animation (Leaf Group)

Here you can adjust the parameters used for animating this group of leaves. In editor mode the wind is always coming from the same direction, and wind zones are not active. If you select too high values for Main Wind and Main Turbulence the leaves may float away from the branches.

Main WindPrimary wind effect. Usually this should be kept to zero or low values for leaves, to avoid them floating away from the parent branch.
Main TurbulenceSecondary turbulence effect. Usually this should be kept to a low values for leaves.
Edge TurbulenceDefines how much wind turbulence occurs along the edges of the leaves.
