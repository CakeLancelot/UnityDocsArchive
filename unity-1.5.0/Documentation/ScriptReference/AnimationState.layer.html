AnimationState.layer

int layer

The layer of the animation. When calculating the final blend weights, animations in higher layers will get their weights distributed first. Lower layer animations only receive blend weights if the higher layers didn't use up all blend weights.

// Puts the walk and run animation in layer 1 animation["Walk"].layer = 1;
animation["Run"].layer = 1;
