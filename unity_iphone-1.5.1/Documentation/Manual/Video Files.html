Movie Texture

Movie Textures are not supported on iPhone. Instead a full-screen streaming playback is provided using iPhoneUtils.PlayMovie and iPhoneUtils.PlayMovieURL.

In order to support streaming playback movie files must be kept in the original format and therefore their import process differ from usual Unity assets.


Video files should be stored in the iPhoneStreamingAssets folder

Alternatively you can specify a network-based URL to play a movie located on a remote server.

Unity iPhone supports any movie that already play correctly on an iPod or iPhone, usually that means files with the extensions .mov, .mp4, .mpv, and .3gp and using one of the following compression standards:

  • H.264 Baseline Profile Level 3.0 video
  • MPEG-4 Part 2 video

For more information about supported compression standards consult with iPhone SDK MPMoviePlayerController Class Reference.

As soon as you call iPhoneUtils.PlayMovie or iPhoneUtils.PlayMovieURL, the screen will fade from your current content to the designated background color. It might take some time while movie is ready for playing, meanwhile the player will continue displaying the background color and may also display a progress indicator to let the user know the movie is loading. When playback finishes, the screen will fade back to your content.

Details

During build player step Unity will scan iPhoneStreamingAssets folder for movies. If suitable movie files are found, they will be stored in your application’s bundle along with other resources. Use iPhoneUtils.PlayMovie to play them.

Playing the Movie

First create a new folder called iPhoneStreamingAssets just next to your project's Assets Folder. Place movie files in that (iPhoneStreamingAssets) folder.

For instance we place StarWars.mp4 into iPhoneStreamingAssets folder and attach the following script to initiate movie playback once application launches: 

function Start () {
	iPhoneUtils.PlayMovie("StarWars.mp4", Color.black);
}

For more information about playing movies, see the Script Reference page.
