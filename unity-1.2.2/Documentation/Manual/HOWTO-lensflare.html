How do I make a Lens Flare

Unity Manual > HOW-TOs > Graphics how-tos > How do I make a Lens Flare
  1. Choose GameObject->Create Empty to add an empty game object to the scene.
  2. Add a lens flare component by choosing Component -> Rendering -> Lens Flare.
  3. In the inspector, select a lens flare to render using the pop menu.

  1. To see the effects in a scene view, make sure the FX button is checked in the scene view toolbar:

Hints

  • To design your own lensflares, you need to create some lens flare Assets. Start by duplicating some of the ones we provided in the the Lens Flares folder of the Standard Assets. Then modify from that.
