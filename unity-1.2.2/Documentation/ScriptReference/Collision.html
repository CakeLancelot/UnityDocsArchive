Collision.relativeVelocity

Vector3 relativeVelocity

The relative linear velocity of the two colling game object's

// Play a sound when we hit an object with a big velocity
function OnCollisionEnter(collision : Collision) {
    if (collision.relativeVelocity.magnitude > 2)
        audio.Play();
}

Collision.rigidbody

Rigidbody rigidbody

The rigidbody we hit. This is null if the object we hit is a collider without rigid body.

// Make all rigidbodies we touch fly upwards
function OnCollisionStay(collision : Collision) {
    // Check if the collider we hit has a rigidbody
    // Then apply the force
    if (collision.rigidbody) {
        collision.rigidbody.AddForce (Vector3.up * 15);
    }
}

Collision.collider

Collider collider

The collider we hit. This is null if the object we hit is a rigidbody. To find all colliders that were hit in detail you have to iterate the contact points Collision.contacts.

Collision.transform

Transform transform

The transform of the object we hit.

If we collided against a rigidbody with a collider, the transform will be the transform attached to the rigidbody. If we collided against a collider without a rigidbody, the transform will be the transform attached to the collider.

Collision.other

Component other

other is either the rigidbody or the collider that we are colliding with. other is never null.

Collision.contacts

ContactPoint[] contacts

The contact points that were used to keep the two objects separate! Every contact point contains a contact point, normal and the two colliders that collided.

function OnCollisionStay(collision : Collision) {
    // Check if the collider we hit has a rigidbody
    // Then apply the force
    for (var contact : ConactPoint in collision.contacts) {
        print(contact.thisCollider.name + " hit " + contact.otherCollider.name);
        // Visualize the contact point
        Debug.DrawRay(contact.point, contact.normal, Color.white);
    }
}
