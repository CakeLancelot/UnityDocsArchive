Collision.transform

Transform transform

The Collider we hit (Read Only). This is null if the object we hit is a rigidbody. To find all colliders that were hit in detail you have to iterate the contact points (contacts property). The Transform of the object we hit (Read Only).

If we collided against a collider with a Rigidbody, the transform will be the transform attached to the rigidbody. If we collided against a collider without a rigidbody, the transform will be the transform attached to the collider.
