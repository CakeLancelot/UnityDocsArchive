This is an alternative to assigning lens flares to light sources. You can just create lens flares in the level...

Variables

LensFlare.flare

Flare flare

The lens flare asset to use

LensFlare.brightness

float brightness

The strength of the flare. This controls the size & brightness of the flare elements.

LensFlare.color

Color color

The Color of the lens flare. This controls the color of some flare elements (if the FlareElement has use light color enabled).
