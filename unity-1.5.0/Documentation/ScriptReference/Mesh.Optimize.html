Mesh.Optimize

void Optimize ()

Optimizes the mesh for display. This operation might take a while but will make the geometry displayed be faster. For example it generates triangles strips out of the triangles.

function Start ()
{
var mesh : Mesh = GetComponent(MeshFilter).mesh;
mesh.Optimize();
}
