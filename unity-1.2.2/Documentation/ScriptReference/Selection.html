class Selection
Overview | Class Hierarchy | Alphabetical Reference

This is an editor class. To use it you have to place your script in Assets/Editor inside your project folder. Editor classes are in the UnityEditor namespace so you need to add "import UnityEditor;" in the beginning of the script.

Access to the selection in the editor.

Class Variables

Selection.transforms

Transform[] transforms

Returns the top level selection, excluding prefabs. This is the most common selection type when working with scene objects.

Selection.activeTransform

Transform activeTransform

Returns the active transform. (The one shown in the inspector) This will never return prefabs or non-modifyable objects.

Selection.gameObjects

GameObject[] gameObjects

Returns the actual game object selection. Includes prefabs, non-modifyable objects. When working with objects that are primarily in a scene, it is strongly recommended to use Selection.transforms instead.

Selection.activeGameObject

GameObject activeGameObject

Returns the active game object. (The one shown in the inspector) It will also return game objects that might be prefabs or non-modifyable objects.

Selection.activeObject

Object activeObject

Returns the actual object selection. Includes prefabs, non-modifyable objects. When working with objects that are primarily in a scene, it is strongly recommended to use Selection.activeTransform instead.

Selection.objects

Object[] objects

The actual unfiltered selection. You can also assign objects to the selection

Class Methods

Selection.GetTransforms

Transform[] GetTransforms (SelectionMode mode)

Allows for fine grained control of the selection type using the SelectionMode bitmask.

Selection.GetFiltered

Object[] GetFiltered (Type type, SelectionMode mode)

Returns the current selection filtered by type and mode. Only objects derived from type will be returned. if type is a subclass of Component or GameObject the full SelectionMode is supported. if type does not subclass from Component or GameObject (eg. LodMesh or ScriptableObject) only SelectionMode.ExcludePrefab, SelectionMode.Editable.

import UnityEditor;
// MenuItem adds a menu item in the GameObject menu
// and executes the following function when clicked
@MenuItem ("GameObject/Apply values")
static function ApplyHardcodedValues ()
{
    // Get all selected game objects that we are allowed to modify!
    var gos = Selection.GetFiltered(GameObject, SelectionMode.Editable);
    // Change the values of all
    for (var go in gos)
    {
        // Change the layer of the game object to 2
        go.layer = 2;
    }


    // Get all selected rigid bodies that we are allowed to modify!
    var bodies = Selection.GetFiltered(Rigidbody, SelectionMode.Editable);
    // Change the values of all
    for (var body in bodies)
    {
        // Change the layer of the game object to 2
        body.mass = 5;
    }
}
