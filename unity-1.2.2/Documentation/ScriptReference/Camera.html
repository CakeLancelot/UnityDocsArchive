A screen space point is defined in pixels. The left-bottom of the screen is (0,0). The right-top is (screenWidth,screenHeight) The z position is between 0...1. 0 is on the near plane. 1 is on the far plane

A viewport space point is normalized and relative to the camera The left-bottom of the camera is (0,0). The top-right is (1,1) The z position is between 0...1. 0 is on the near plane. 1 is on the far plane

A world space point is defined in global coordinates (eg. Transform.position)

The top/bottom clipping plance distance when in ortho graphic mode

bool isOrthoGraphic

Is the camera ortho graphic or perspective?

Camera.depth

float depth

Depth in the camera rendering order. The camera with the lowest depth is rendered first.

Camera.aspect

float aspect

The aspect ratio of the camera (Read Only). The aspect ratio is always the same as the screen's aspect ratio. FIXME: is this also the case if we have a non-fullscreen camera?

Camera.cullingMask

int cullingMask

This is used to render parts of the scene selectively. If the rendered GameObject's layerMask AND the Camera's cullingMask is zero the game object will be invisible from this camera.

Camera.backgroundColor

Color backgroundColor

The color with which the screen will be cleared if clear flags are set to clear color.

Rect rect

Where on screen is the camera rendered in normalized coordinates.

Camera.pixelRect

Rect pixelRect

Where on screen is the camera rendered in pixel coordinates

Camera.targetTexture

RenderTexture targetTexture

Where on screen is the camera rendered in pixel coordinates

Camera.pixelWidth

float pixelWidth

How wide is the camera, in pixels.

Camera.pixelHeight

float pixelHeight

How tall is the camera, in pixels.

Camera.cameraToWorldMatrix

Matrix4x4 cameraToWorldMatrix

The camera-to-world matrix (Read Only). Use this to calculate where in the world as specific point is in camera space.

Camera.worldToCameraMatrix

Matrix4x4 worldToCameraMatrix

The world-to-camera matrix. Use this to calculate the camera space position of objects. If you change this matrix, the camera no longer updates its rendering based on its postion. This lasts until you call ResetWorldToCameraMatrix.

Camera.projectionMatrix

Matrix4x4 projectionMatrix

Set a custom projection matrix. If you change this matrix, the camera no longer updates its rendering based on its field-of-view. This lasts until you call ResetProjectionMatrix.

Camera.velocity

Vector3 velocity

Get the world-space speed of the camera over the last frame (Read Only).

Camera.clearFlags

CameraClearFlags clearFlags

The bitmask determining what to clear: CameraClearFlags.Color, CameraClearFlags.Depth, CameraClearFlags.Nothing

Methods

Camera.ResetWorldToCameraMatrix

void ResetWorldToCameraMatrix ()

Make the rendering position reflect the camera's position in the scene. Call this to undo the effect of setting worldToCameraMatrix.

void ResetProjectionMatrix ()

Make the rendering position reflect the camera's position in the scene. Call this to undo the effect of setting projectionMatrix.

Camera.ViewportPointToRay

Ray ViewportPointToRay (Vector3 position)

Returns a ray in world space starting at the near plane of the camera at position. Any point on the ray corresponds to /position's/'s x, y coordinates on the screen. (viewPoint.z is ignored)

Camera.ScreenPointToRay

Ray ScreenPointToRay (Vector3 position)

Returns a ray in world space starting at the near plane of the camera at position. Any point on the ray corresponds to /position's/'s x, y coordinates on the viewport. (position.z is ignored)

Messages Sent

Camera.OnPreCull

void OnPreCull ()

OnPreCull is called before a camera culls the scene. This message is sent to all scripts attached to the camera.

Camera.OnPreRender

void OnPreRender ()

OnPreRender is called before a camera starts rendering the scene. This message is sent to all scripts attached to the camera.

Camera.OnPostRender

void OnPostRender ()

OnPostRender is called after a camera finished rendering the scene. This message is sent to all scripts attached to the camera.

Camera.OnRenderImage

void OnRenderImage (RenderTexture source, RenderTexture destination)

OnRenderImage is called after rendering is complete. It allows you to do image based effects by blitting from one render texture into another with shader based filters. This message is sent to all scripts attached to the camera.

Class Variables

Camera.main

Camera main

The first enabled camera tagged "MainCamera". Returns null if there is no camera in the scene. (Read Only)

Camera.current

Camera current

The camera we are currently rendering with. Most of the time you will want to use Camera.main instead. Use this function only when implementing one of the following events: MonoBehaviour.OnRenderImage, MonoBehaviour.OnPreRender, MonoBehaviour.OnPostRender

Camera.allCameras

Camera[] allCameras

All enabled cameras in the scene.
