Debug.DrawRay

void DrawRay (Vector3 start, Vector3 dir, Color color = Color.white)

Draws a line from start to start + dir with color.

// Draws a 10 meter long green line from the position // forward along the transform's z-axis. function Update () {
var forward = transform.TransformDirection(Vector3.forward) * 10;
Debug.DrawRay (transform.position, forward * 10, Color.green);
}
