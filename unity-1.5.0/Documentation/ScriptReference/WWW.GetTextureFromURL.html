WWW.GetTextureFromURL

Texture2D GetTextureFromURL (string url)

Downloads an image from the web page and generates a FileTexture from it. The image must be in JPG format.

The function will block the execution of the game until the data has been downloaded. The image dimensions must be a power of two in each direction or this function will pad the texture with white pixels up to the next power of two size.
