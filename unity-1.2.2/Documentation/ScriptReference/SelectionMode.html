enum SelectionMode

SelectionMode can be used to tweak the selection returned by Selection.GetTransforms.

Values

Unfiltered

TopLevel

Deep

ExcludePrefab

Editable

The default transform selection mode is: SelectionMode.TopLevel | SelectionMode.ExcludePrefab | SelectionMode.OnlyUserModifyable

Values

SelectionMode.Unfiltered

The actual selection

SelectionMode.TopLevel

Only return the top most selected transform. A selected child of another transform which is also selected will be filtered out.

SelectionMode.Deep

Also Returns all child transforms of the selection

SelectionMode.ExcludePrefab

Excludes any prefabs from the selection

SelectionMode.Editable

Excludes any objects which shall not be modified. Eg. This will filter generated prefabs from an fbx file but not normal prefabs.
