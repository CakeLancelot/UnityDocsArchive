class CapsuleCollider

Collider

A capsule-shaped primitive collider.

Variables

center

radius

height

direction

Capsules are cylinders with a half-sphere at each end.

See Also: BoxCollider, SphereCollider, DynamicsMaterial, RigidBody

Variables

CapsuleCollider.center

Vector3 center

The center of the capsule, measured in the object's local space.

// Reset the center to the transform's position
collider.center = Vector3.zero;

CapsuleCollider.radius

float radius

The radius of the sphere, measured in the object's local space. The capsule's radius will be scaled by the transform's scale.

collider.radius = 1;

CapsuleCollider.height

float height

The height of the capsule meased in the object's local space. The capsule's height will be scaled by the transform's scale. Note that the height is the actual height including the half-spheres at each end.

collider.height = 5;

CapsuleCollider.direction

int direction

The direction of the capsule. 0 -> Capsule height is along the x-axis. 1 -> Capsule height is along the y-axis. 2 -> Capsule height is along the z-axis.

// Make the capsule height be along the x-axis
collider.direction = 0;
