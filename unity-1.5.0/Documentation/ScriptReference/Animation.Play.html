Animation.Play

bool Play (PlayMode mode = PlayMode.StopSameLayer)

Plays the default animation. The animation will be played abruptly without any blending.

if mode is PlayMode.StopSameLayer all animations in the same layer will be stopped. if mode is PlayMode.StopAll all animations currently playing will be stopped.

If the animation is already playing, other animations will be stopped but the animation will not rewind to the beginning.

If the animation is not set to be looping it will be stopped and rewinded after playing.

// Plays the default animation animation.Play();

bool Play (string animation, PlayMode mode = PlayMode.StopSameLayer)

Plays the animation with name animation. The animation will be played abruptly without any blending.

if mode is PlayMode.StopSameLayer all animations in the same layer will be stopped. if mode is PlayMode.StopAll all animations currently playing will be stopped. If the animation is already playing, other animations will be stopped but the animation will not rewind to the beginning.

If the animation is not set to be looping it will be stopped and rewinded after playing.

// Plays the walk animation - stops all other animations in the same layer animation.Play("walk");

// Plays the walk animation - stops all other animations animation.Play("walk", PlayMode.StopAll);
