enum ParticleRenderMode
Overview | Class Hierarchy | Alphabetical Reference

The rendering mode for particle systems.

Values

Billboard

Stretch

SortedBillboard

This is used by the ParticleRenderer to determine how to render the particles

Values

ParticleRenderMode.Billboard

Render the particles as billboards facing the player The default mode

ParticleRenderMode.Stretch

Stretch particles in the direction of motion

ParticleRenderMode.SortedBillboard

Sort the particles back-to-front. This looks better with blended particle shaders, but does come at a higer price.
