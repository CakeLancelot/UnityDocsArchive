An event.

Variables
mousePosition

The mouse position.

delta

The relative movement of the mouse compared to last event.

button

Which mouse button was pressed.

clickCount

How many consecutive clicks have we recieved

character

The character pressed when EventType is keyup or keydown.

keyCode

The raw key code for keyboard events. This will return an enum that matches the physical key pressed.

shift

Is Shift held down? (Read Only)

control

Is Control key held down? (Read Only)

alt

Is Alt/Option key held down? (Read Only)

command

Is command key held down? (Read Only)

capsLock

Is Caps Lock on? (Read Only)

numeric

Is the current keypress on the numeric keyboard? (Read Only)

functionKey

Is the current keypress a function key.

isKey

Is this event a keyboard event?

isMouse

Is this event a mouse event?
