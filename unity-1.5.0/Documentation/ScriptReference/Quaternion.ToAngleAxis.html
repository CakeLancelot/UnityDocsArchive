Quaternion.ToAngleAxis

void ToAngleAxis (out float angle, out Vector3 axis)

Converts a rotation to angle-axis representation.

// Extracts the angle - axis rotation from the transform rotation var angle = 0.0;
var axis = Vector3.zero;
transform.rotation.ToAngleAxis(angle, axis);
