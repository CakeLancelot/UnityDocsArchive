class AudioClip
Overview | Class Hierarchy | Alphabetical Reference

Object

A containter for audio data.

Variables

length

An AudioClip stores the audio file either compressed as ogg vorbis or uncompressed. AudioClips are referenced and used by AudioSources to play sounds.

See Also: AudioClip in the Components Reference
