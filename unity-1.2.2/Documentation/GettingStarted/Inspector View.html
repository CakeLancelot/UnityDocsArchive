The Inspector

Getting Started > The Inspector

Positioning objects in a scene is nice and well, but you need to edit the details of them in order to make them fun.

In order to do this, you use the inspector. The inspector tracks the currently selected object and shows the settings for that object in detail. This is the place where you fine tune the appearance and add functionality to any objecs you have in the scene.

The Structure of Objects

The entities of a Unity game are called GameObjects, or a GO for short. A GO is little more than a position and container for components. Components and are what sets your games alive with action. Components frequently cooperate as in the collision components telling the other physic components when they hit stuff.

There are many types of components, and they are added to your game objects with the 'Component' menu:

There are a lot of them, so don't try and learn them all at once!

Keep Going

Zoom out the scene view and enable in-game lighting by pressing the light icon in the toolbar at the top of the scene view:

Select the light in the top-left corner of the screen. The inspector changes to show the settings for this object.

Click on the color of the light and change it a bit around. Notice how the scene view changes mood as you change the lighting.

Above the light setting, there is a small question mark. Click this to get on-line help for this component. Use this to learn about the Unity features at your own pace.
