Font

Components > Assets > Font

Defines a font for drawing using either the GUI Text the Text Mesh components.

Properties

Property:Function:
Ascii Start Offset The ASCII code of the first character in the font.
Font Count X The number of characters each line in the font texture.
Font Count Y Number of lines in the font texture.
Kerning Kerning of spacing between characters (less than 1.0 pulls characters together, larger values increase the spacing)
Line Spacing Spacing between lines as multiple of the height of the characters
Per Character Kerning Used for applying different kerning values per character.
Convert Case Defines how character case is handled. Useful if the font only contains a single case of characters.
Default Material The material used to render the font, unless overridden by the GUI Text compontent.

Details

The font glyphs themselves come from a grid of characters inside a texture. This Texture should be assigned to the material used as Default Material.
