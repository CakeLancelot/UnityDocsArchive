enum SendMessageOptions
Overview | Class Hierarchy | Alphabetical Reference

Options for how to send a message.

Values

RequireReceiver

DontRequireReceiver

This is used by SendMessage & BroadcastMessage in GameObject & Component.

Values

SendMessageOptions.RequireReceiver

A receiver is required. If none is found, an error is printed to the console. (This is the default value.)

SendMessageOptions.DontRequireReceiver

No receiver is required.
