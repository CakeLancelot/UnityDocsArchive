This is a small utility module for retrieving URLs.

You start a download in the background by calling WWW(url) which returns a new WWW object.

You can inspect the isDone property to see if the download has completed or yield the download object to automatically wait until it is (without blocking the rest of the game).

Useful if you want to get some data from a web server for integration with a game such as highscore lists or calling home for some reason. There is also funtionality to create textures from images downloaded from the web.

// This example assumes that the user has entered his name into a
// variable called name and that score contains the user's current
// score.

var highscore_url="http://www.my-site.com/highscores.pl?game=GameName";
// add a name parameter to the URL:
highscore_url += "&name=" + WWW.EscapeURL(name);
// add the score
highscore_url += "&score=" + score;

// Post the URL to the site and create a download object to get the result.
download=WWW(highscore_url);

yield download; // Wait until the download is done
if(download.error) {
    print("There was an error downloading the URL: " + download.error);
}
else {
    /* show the highscores */
}
// The implementation of highscores.pl is left as an exercise for the
// reader. :-)

Variables

WWW.data

string data

Returns the contents of the fetched URL as string.

If the object has not finished downloading the data, it will block until the download has finished. Use isDone or yield to see if the data is available.

WWW.size

int size

Returns the size of the fetched URL.

If the object has not finished downloading the data, it will block until the download has finished. Use isDone or yield to see if the data is available.

WWW.error

string error

Returns an error string if there was an error during the download. If there was no error, error will return null.

If the object has not finished downloading the data, it will block until the download has finished. Use isDone or yield to see if the data is available.

WWW.texture

FileTexture texture

Returns a FileTexture generated from the downloaded data.

If the object has not finished downloading the data, it will block until the download has finished. Use isDone to see if the data is available.

The image dimensions must be a power of two in each direction or the texture will get padded with white pixels up to the next power of two size.

Note: This class only supports RGB jpeg images.

    // Get the latest webcam shot from outside "Friday's" in Times Square
    var url="http://images.earthcam.com/ec_metros/ourcams/fridays.jpg"
    private var www : WWW;
    www=WWW(url); // Start a download of the given URL
    
    while(true) {
        if( www.isDone ) { // check whether the download is done
            // assign the downloaded image as a texture to the object
            renderer.material.mainTexture=www.texture;
            
            // wait for 20 seconds
            yield WaitForSeconds(20);
            
            // then start a new download
            www = new WWW(url);
        }
        // wait for a second before checking again
        yield WaitForSeconds(1);
    } *

WWW.isDone

bool isDone

isDone is false until the object has finished downloading the url.

if you try to access any data while isDone is false, the program will block while finishing the download.

Methods

WWW.LoadImageIntoTexture

void LoadImageIntoTexture (FileTexture tex)

Replaces the contents of a FileTexture with an image from the downloaded data.

If the object has not finished downloading the data, it will block until the download has finished. Use isDone to see if the data is available or yield using the WWW object to automatically wait until it is.

The image dimensions must be a power of two in each direction or the texture will get padded with white pixels up to the next power of two size.

Note: This class only supports RGB jpeg images.

    // Get the latest webcam shot from outside "Friday's" in Times Square
    var url="http://images.earthcam.com/ec_metros/ourcams/fridays.jpg";
    private var www : WWW;
    
    while(true) {
        www=WWW(url); // Start a download of the given URL
        yield www; // wait until the download is done
        
        // assign the downloaded image to the main texture of the object
        www.LoadImageIntoTexture(renderer.material.mainTexture);
            
        // wait for 20 seconds
        yield WaitForSeconds(20);
    }

Class Methods

WWW.EscapeURL

string EscapeURL (string s)

Replaces illegal characters in s with the correct URL escape code. Useful when building url parameters.

    // This will print the string "Testing%201%2C2%2C3" to the console
    print(WWW.EscapeURL("Testing 1,2,3"));

WWW.UnEscapeURL

string UnEscapeURL (string s)

The inverse of WWW.EscapeURL. Replaces any occurences of %XX with the corresponding letter.

// This will print the string "Testing 1,2,3" to the console
print(WWW.UnEscapeURL("Testing%201%2C2%2C3"));

WWW.GetURL

string GetURL (string url, bool includeHeader=false)

Fetches the address stored in url, can fetch both http and ftp URLs. The entire data is returned in a string.

The function will block the execution of the game until the data has been downloaded. If includeHeader is set to true, the returned string will also include the HTTP response headers from the web server.

Any parsing of the returned data is up to you.

    otees_home_page=WWW.GetURL("http://www.otee.dk/");

WWW.GetTextureFromURL

FileTexture GetTextureFromURL (string url)

Downloads an image from URL and generates a FileTexture from it.

The function will block the execution of the game until the data has been downloaded. The image dimensions must be a power of two in each direction or this function will pad the texture with white pixels up to the next power of two size.

Note: This function only supports RGB jpeg images.
