Joint.connectedBody

Rigidbody connectedBody

A reference to another rigid body this joint connects to. If not set, the joint connects the object to the world.

// Connects the joint to the world instead of another rigid body hingeJoint.connectedBody = null; *
// Connects the joint to another body var otherBody : Rigidbody;
hingeJoint.connectedBody = otherBody;
