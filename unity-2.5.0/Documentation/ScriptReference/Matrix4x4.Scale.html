static function Scale (v : Vector3) : Matrix4x4

Description

Creates a scaling matrix.

Returned matrix is such that scales along coordinate axes by a vector v. The matrix looks like this:

v.x 0 0 0
0 v.y 0 0
0 0 v.z 0
0 0 0 1

See Also: TRS function.
