Collider.ClosestPointOnBounds

Vector3 ClosestPointOnBounds (Vector3 position)

The closest point to the bounding box of the attached collider. This can be used to calculate hit points when applying explosion damage.

var hitPoints = 100.0;
function ApplyHitPoints (explosionPos : Vector3, radius : float) {
// The distance from the explosion position to the surface of the rigidbody var closestPoint = collider.ClosestPointOnBounds(explosionPos);
var distance = Vector3f.Distance(closestPoint, explosionPos);

// The hit points we apply fall decrease with distance from the hit point var hitPoints = 1.0 - Mathf.Clamp01(distance / radius);

// This is the final hitpoints we want to apply. 10 at maximum hitPoints *= 10;
}
