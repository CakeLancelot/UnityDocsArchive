class AddComponentMenu
Overview | Class Hierarchy | Alphabetical Reference

Attribute

The AddComponentMenu attribute allows you to place a script anywhere in the "Component" menu, instead of just the "Component->Scripts" menu.

Constructors

AddComponentMenu

You use this to organize the Component menu better, this way improving workflow when adding scripts. Important notice: You need to restart

// C# example:
[AddComponentMenu("Transform/Follow Transform")]
class FollowTransform : MonoBehaviour
{
}

Constructors

AddComponentMenu.AddComponentMenu

AddComponentMenu (string menuName)

The script will be placed in the component menu according to menuName. menuName is the path to the component "Rendering/DoSomething" if menuName is "" the component will be hidden from the menu. if menuName is "" the component will be hidden from the menu.
