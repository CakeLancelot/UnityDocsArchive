enum FilterMode
Overview | Class Hierarchy | Alphabetical Reference

Filtering mode for textures. Corresponds to the settings in a Texture's inspector.

Values

Point

Bilinear

Trilinear

This is mainly used when creating textures from code (using RenderTexture).

Values

FilterMode.Point

Point filtering - this has no smoothing.

FilterMode.Bilinear

Bilinear filtering - texture samples are averaged

FilterMode.Trilinear

Trilinear filtering - used for blending between texture mip levels. RenderTextures do not support mipmaps, so this setting reverts to Bilinear for those.
