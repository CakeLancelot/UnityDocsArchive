class ControllerColliderHit

ControllerColliderHit is used by CharacterController.OnControllerColliderHit to give detailed information about the collision and how to deal with it.

Variables

ControllerColliderHit.controller

CharacterController controller

The controller that hit the collider

ControllerColliderHit.collider

Collider collider

The collider that was hit by the controller

ControllerColliderHit.point

Vector3 point

The impact point in world space

ControllerColliderHit.normal

Vector3 normal

The normal of the impact point in world space

ControllerColliderHit.moveDirection

Vector3 moveDirection

The direction the controller was moving before it hit the collider. Note that this can be different the what you pass to CharacterController.Move, because all movement is constrained by collisions.

ControllerColliderHit.moveLength

float moveLength

How far the character has travelled until it hit the collider. Note that this can be different the what you pass to CharacterController.Move, because all movement is constrained by collisions.
