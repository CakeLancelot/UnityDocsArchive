iPhone Scripting

Unity iPhone provides a number of new scripting APIs to access unique device functionality. The following new scripting classes were introduced:

iPhoneInputAccess to multi-touch screen, accelerometer, device orientation and geographical location.
iPhoneSettingsiPhone specific settings, such as screen orientation, dimming and information about device hardware.
iPhoneKeyboardSupport for native on-screen keyboard.
iPhoneUtilsUseful functions for movie playback, anti-piracy protection and vibration.
