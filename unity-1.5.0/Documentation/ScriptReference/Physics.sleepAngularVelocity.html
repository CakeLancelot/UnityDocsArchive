Physics.sleepAngularVelocity

float sleepAngularVelocity

The default angular velocity, below which objects start sleeping (default 0.14). Must be positive. See Rigidbody Sleeping for more information. This can be overridden per rigid body using Rigidbody.sleepAngularVelocity.

This value is usually changed in Edit->Project Settings->Physics inspector instead of from scripts.

Physics.sleepAngularVelocity = 0.1;
