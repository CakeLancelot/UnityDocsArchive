Graphics.supportsVertexProgram

bool supportsVertexProgram

Does this machine support vertex program? (Read Only) Returns true if the machine supports ''hardware'' vertex programs. returns false if not

Class Methods

Graphics.DrawQuad

void DrawQuad (Material material, Vector3 center, float size)

Draw a billboarded quad, with a given material, with all channels available. Use this to draw any billboard, with full bumpmapping. The quad is drawn at the worldpsace position center and has a given size.

Graphics.DrawQuad2

void DrawQuad2 (Vector3 center, float size)

Draw a billboarded quad, with the current active material, with all channels available. Use this to draw any billboard, with full bumpmapping.

Graphics.DrawMesh

void DrawMesh (LodMesh mesh, Material material, Vector3 position, Quaternion rotation)

Draw a mesh at a given position, with a given rotation. This function will assign the material, set up the modelview matrix and draw the object. Any passes defined in the material's Shader will be drawn. The mesh is not affected by scene lights.

Graphics.DrawMesh2

void DrawMesh2 (LodMesh mesh, Vector3 position, Quaternion rotation)

Draw a mesh at a given position, with a given rotation. This function will set up the modelview matrix and draw the mesh.

Graphics.DrawMeshRaw

void DrawMeshRaw (LodMesh mesh)

Draw a mesh. This function draws a mesh. The currently set material and modelview matrix is used.
