How the light is rendered.

Values

Auto

ForcePixel

ForceVertex

Values

LightRenderMode.Auto

Automatically choose whether to render the light as a pixel or vertex light (recommended)

LightRenderMode.ForcePixel

The light is a spot light. Use this only for really important lights, like a player flashlight.

LightRenderMode.ForceVertex

Force the light to be a vertex light - this is good for background lighting.
