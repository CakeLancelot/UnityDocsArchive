static function FreeMoveHandle (position : Vector3, ref rotation : Quaternion, size : float, capFunc : Handles.DrawCapFunction) : Vector3

Parameters

NameDescription
position the position of the handle
rotation the rotation of the handle. This can be changed with raysnapping.
size the size of the handle.
capFunc the function to use for drawing the handle e.g. Handles.DrawRectangle

Description

Make an unconstrained movement handle.

This can move freely in all directions. Hold down CMD to raysnap agains colliders in the scene.
