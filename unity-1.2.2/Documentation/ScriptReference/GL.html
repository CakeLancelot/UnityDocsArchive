GL.SetAppBlendModes

void SetAppBlendModes (int src, int dest)

Set the custom shaderlab blend modes for AppSrcAdd and AppDestAdd blend modes. src gets mapped to AppSrcAdd and dest gets mapped to AppDestBlendMode. After calling this function, you must call Shaders SetPass function to have the changes take effect.

GL.LoadOrtho

void LoadOrtho ()

Helper function to set up an ortho perspective transform. After calling LoadOrtho, the viewing frustum goes from (0,0,-1) to (1,1,100)
