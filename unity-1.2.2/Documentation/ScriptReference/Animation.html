Methods

Animation.Play

bool Play (AnimationPlayMode mode = AnimationPlayMode.Stop)

Play clip property. Returns true if the animation is available. if mode is Stop, it will stop all animations that were started with this component before playing if mode is Queue, it will wait with playing the animation until all other animations have completed playing if mode is Mix, it will only replace animations that animate the same component. This is used to mix animations on separate body parts.

// Plays the default animation and stops
// all animations that were playing before
animation.Play();

// Makes the default animation repeat 3 times.
animation.Play(AnimationPlayMode.Queue);
animation.Play(AnimationPlayMode.Queue);
animation.Play(AnimationPlayMode.Queue);

 
bool Play (string name, AnimationPlayMode mode = AnimationPlayMode.Stop)

Play the clip named name. if mode is StopAll, it will stop all animations that were started with this component before playing if mode is Queue, it will wait with playing the animation until all other animations have completed playing if mode is Mix, it will only replace animations that animate the same component. This is used to mix animations on separate body parts. Returns false if the animation named name couldn't be found.

// Plays the clip named "walk".
animation.Play("walk");
// Queues the run animation to play after the walk
animation.Play("run", AnimationPlayMode.Queue);

Animation.IsPlaying

Is the animation named name playing?

// Plays an animation only if we are not playing it already.
function OnMouseEnter() {
    if (!animation.IsPlaying("mouseOverEffect"))
        animation.Play("mouseOverEffect");
}
