How do I install or upgrade Standard Assets?

Unity iPhone ships with iPhone Standard Assets which contain a first person controller, iPhone specific shaders, projectors, shadows, common scripts and more.

Installing

When creating a new project Unity iPhone automatically installs the iPhone Standard Assets.


The Standard Assets are included in all new Projects by default

Upgrading

Sometimes you might want to upgrade your Standard Assets, for example because a new version of Unity iPhone ships with new iPhone Standard Assets:

  1. Open your project.
  2. Choose Assets->Import Package from the menu.
  1. Select iPhone Standard Assets.unitypackage.
  1. A list of new or replaced assets will be presented, click Import.
