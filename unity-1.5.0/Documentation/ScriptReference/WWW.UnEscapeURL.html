WWW.UnEscapeURL

string UnEscapeURL (string s)

The inverse of WWW.EscapeURL. Replaces any occurences of %XX with the corresponding character.

// This will print the string "Testing 1,2,3" to the console print(WWW.UnEscapeURL("Testing%201%2C2%2C3"));
