This structure is used throughout Unity to pass 3D positions and vectors around. It also contains methods for doing common manipulations on vectors.

Class Methods

Vector3.Lerp

Vector3 Lerp (Vector3 from, Vector3 to, float t)

Interpolates from towards to by t. t is clamped between [0...1] When t = 0 -> returns from When t = 1 -> return to When t = .5 -> returns the average of from and to.

Vector3.Slerp

Vector3 Slerp (Vector3 from, Vector3 to, float t)

Spherically interpolates from towards to by t. The returned Vector3's Magnitude will be interpolated between from and to. t is clamped between [0...1]
