Simple access to web pages.

Members

Variables

Functions

Class Functions

This is a small utility module for retrieving the contents of URLs.

You start a download in the background by calling WWW(url) which returns a new WWW object.

You can inspect the isDone property to see if the download has completed or yield the download object to automatically wait until it is (without blocking the rest of the game).

Use it if you want to get some data from a web server for integration with a game such as highscore lists or calling home for some reason. There is also functionality to create textures from images downloaded from the web.

// This example assumes that the user has entered his name into a // variable called name and that score contains the user's current // score.
var highscore_url="http://www.my-site.com/highscores.pl?game=GameName"; // add a name parameter to the URL: highscore_url += "&name=" + WWW.EscapeURL(name);
// add the score highscore_url += "&score=" + score;

// Post the URL to the site and create a download object to get the result. download=WWW(highscore_url);

yield download; // Wait until the download is done if(download.error) {
print("There was an error downloading the URL: " + download.error);
}
else {
/* show the highscores */
}
// The implementation of highscores.pl is left as an exercise for the // reader :-)

Variables

NameDescription
data

Returns the contents of the fetched web page as a string.

size

Returns the size of the fetched web page data in bytes.

error

Returns an error message if there was an error during the download.

texture

Returns a Texture2D generated from the downloaded data. The data must be an image in JPG format.

isDone

Returns true if the download has already finished.

Functions

NameDescription
LoadImageIntoTexture

Replaces the contents of an existing Texture2D with an image from the downloaded data. The data must be an image in JPG format.

Class Functions

NameDescription
EscapeURL

Replaces illegal characters in s with the correct URL escaped code.

UnEscapeURL

The inverse of WWW.EscapeURL. Replaces any occurences of %XX with the corresponding character.

GetURL

Fetches the contents of web page url. Supports both HTTP and FTP addresses.

GetTextureFromURL

Downloads an image from the web page and generates a FileTexture from it. The image must be in JPG format.
