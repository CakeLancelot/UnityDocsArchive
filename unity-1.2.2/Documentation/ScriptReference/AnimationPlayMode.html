enum AnimationPlayMode
Overview | Class Hierarchy | Alphabetical Reference

Used by Animation's Play method.

Values

Stop

Queue

Mix

Values

AnimationPlayMode.Stop

Will stop all animations that were started with this component before playing

AnimationPlayMode.Queue

Will wait with playing the animation until all other animations have completed playing

AnimationPlayMode.Mix

Will only replace animations that animate the same component. This is used to mix animations on separate body parts.
