Skybox

Components > Rendering > Skybox

Skyboxes are rendered before the main scene in order to give the impression of complex scenery at the horizon. They are a box of 6 textures, one for each primary direction.

Properties

Property:Function:
Material The material used to render the skybox. This material should have the Skybox shader set, and each of the materials textures should be set up so it renders the correct texture.

Hints

  • If you have a skybox assigned to a camera, you disable both Color & Depth in camera's clear flags.
