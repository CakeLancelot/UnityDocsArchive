Shader.isSupported

bool isSupported

Can this shader run on the end-users graphics card? This returns true, if the shader itself or any fallbacks setup in the shader are supported.

Class Methods

Shader.ClearAll

void ClearAll ()

Remove all shaders from memory, causing a recompile. This will cause all shaders to be reloaded. Used for debugging shaders from a player application.

Shader.SetColor

void SetColor (string propertyName, Color color)

Sets a global shader color property. This is read if the shader does not define a value for a property. Typically used to offload complex calculation to a script: Do the calculation from scripting, and just read the color.
