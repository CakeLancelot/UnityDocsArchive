Object.FindObjectsOfType

Object[] FindObjectsOfType (Type type)

Returns a list of all active loaded objects of Type type. It will return no assets (meshes, textures, prefabs, ...) or inactive objects.

// When clicking on the object, it will disable all springs on all // hinges in the scene. function OnMouseDown () {
hinges = FindObjectsOfType (HingeJoint);
for (var hinge : HingeJoint in hinges) {
hinge.useSpring = false;
}
}
