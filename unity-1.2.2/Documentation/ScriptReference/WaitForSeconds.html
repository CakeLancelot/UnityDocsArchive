YieldInstruction

WaitForSeconds should be returned using yield in coroutines.

Constructors

WaitForSeconds

It will make the coroutine halt for the amount of seconds specified in the constructor.

// Prints 0
print (Time.time);
// Waits 5 seconds
yield new WaitForSeconds (5);
// Prints 5.0
print (Time.time);
