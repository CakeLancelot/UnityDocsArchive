Meshes

Unity Manual > User Guide > Graphics > Meshes

At the core of any 3D game is Meshes - objects consisting of triangles, with textures applied.

Meshes in Unity are passed through pipelines. Although there are many variations, almost all mesh pipelines have a few things in common:

<!-- img src="http://otee.dk/TODO: graph out this process -->

  • They have a LodMeshFilter at the beginning. This filter loads a mesh from a Mesh asset and inserts it into the pipeline.
  • They have a MeshRenderer at the end. This filter takes the Mesh from in the pipeline and renders it at the Game Object's position. The MeshRenderer controls the object's rendered appearance through its Material.

Mesh Filter

The Mesh Filter takes a mesh from your assets and inserts it into the pipeline.

Property:Function:
Lod Mesh Pointer to a mesh that is inserted into the pipeline. The mesh is stored in your project.

To see the mesh in your scene, add a mesh renderer to the game object.

Mesh Renderer

The Mesh Renderer takes the geometry passed through the Pipeline and renders it at the position defined by the object's Transform component

Properties

Property:Function:
Material A list of materials. This points to the material that defines all rendering properties.
