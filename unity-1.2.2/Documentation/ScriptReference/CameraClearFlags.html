enum CameraClearFlags
Overview | Class Hierarchy | Alphabetical Reference

Enum for setting to Camera.clearFlags, determining what to clear when rendering a Camera. The values can be OR'ed together.

Values

Skybox

Color

Depth

Nothing

Values

CameraClearFlags.Skybox

Clear with the skybox

CameraClearFlags.Color

Clear the color values

CameraClearFlags.Depth

Clear the depth value

CameraClearFlags.Nothing

Don't clear anything
